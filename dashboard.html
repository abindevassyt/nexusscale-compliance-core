State Machine
🟢 CLOSED (Normal)
All MCP calls go through normally. Failures are counted. When failure_count ≥ threshold (5), transitions to OPEN.
🔴 OPEN (Fast-Fail)
All calls immediately raise MCPCircuitOpenError → HTTP 503. After 30s recovery timeout, transitions to HALF_OPEN.
🟡 HALF_OPEN (Probe)
One probe call allowed. Success × 2 → CLOSED. Failure → back to OPEN. Prevents thundering herd on recovery.
